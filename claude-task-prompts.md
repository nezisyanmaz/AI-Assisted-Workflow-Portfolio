# Claude Task Prompts

This file stores example prompts for using Claude with this repository.

## Generate the next content package

Review the content calendar, topic backlog and trend notes in this repository. Choose the next best topic and create a full short-form content package including a script, caption, image prompts, video prompts and CTA.

## Create a weekly content plan

Review the trend notes, content pillars and topic backlog. Create a weekly content plan with 5 short-form video ideas, hooks, suggested visuals and monetisation angles.

## Review performance and suggest next steps

Review the post-performance notes and identify what themes, hooks or formats performed best. Suggest 5 follow-up content ideas and explain why they may work.

## Suggest monetisation opportunities

Review the current content themes and suggest relevant resource page ideas, affiliate link opportunities and simple digital product concepts that fit the audience needs.

## Improve the workflow system

Review the repository structure and suggest how to make this AI-assisted content workflow more efficient, repeatable and useful for managing a small content business.
