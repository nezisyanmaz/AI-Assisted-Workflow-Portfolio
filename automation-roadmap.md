# Automation Roadmap

This file tracks the automation workflows I want to learn and build for an AI-assisted content business.

## 1. Weekly content planning

Goal: Claude reviews trend notes, topic backlog and content pillars, then suggests a weekly content plan.

Future output:

- 5 short-form video ideas
- Hooks
- Script angles
- Suggested visuals
- CTA options
- Monetisation angles

## 2. New topic to content package

Goal: when a new topic is added, Claude helps turn it into a full production package.

Future output:

- Short-form video script
- Caption draft
- Image prompts
- Video prompts
- Talking-avatar direction
- Resource or affiliate link idea

## 3. Post-performance review

Goal: after a post is published, performance notes are reviewed to improve the next content ideas.

Inputs:

- Views
- Saves
- Comments
- Shares
- Link clicks
- Audience questions

Future output:

- Follow-up video ideas
- Better hooks
- Improved CTAs
- New topic angles

## 4. Weekly monetisation review

Goal: Claude reviews content themes and suggests relevant monetisation opportunities.

Future output:

- Resource page ideas
- Affiliate link opportunities
- Digital product ideas
- Lead magnet ideas
- Follow-up content sequences
