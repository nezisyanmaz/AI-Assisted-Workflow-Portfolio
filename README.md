# Claude Content Automation Lab

A learning project for building a GitHub-based AI content operations system using Claude, ChatGPT and creative AI tools.

The goal is to use GitHub as the central workspace for organising content ideas, trend research, scripts, prompts, publishing notes and monetisation plans, while learning how Claude can support and eventually automate parts of the workflow.

## What this repository is for

- Tracking content ideas and trend notes
- Creating short-form video scripts
- Building image and video prompt templates
- Organising content-to-resource and monetisation ideas
- Learning how Claude and GitHub can support content workflow automation

## Current focus

This project is currently focused on documentation, templates and workflow design rather than software engineering.

## Planned automation ideas

- Weekly content planning from trend notes and topic backlog
- New topic to script, caption and visual prompts
- Post-performance review and follow-up content suggestions
- Weekly affiliate, resource page and digital product idea review

## Tool stack

- Claude for workflow review and content planning
- ChatGPT for strategy, scripts and prompt development
- HeyGen for talking-avatar workflows
- Kling for image-to-video prompts
- Canva for visuals and simple digital resources
- CapCut for editing and subtitles
- Beacon for resource pages and digital product delivery
- GitHub for documentation and workflow organisation

## Repository structure

- `automation-roadmap.md` — planned automation workflows I want to learn and build
- `claude-task-prompts.md` — prompts for using Claude with this repository
- `tool-stack.md` — tools used for content, automation and monetisation workflows

## Note

This is an early-stage learning project. It contains workflow documents, templates and prompt structures only. No confidential data is included.
